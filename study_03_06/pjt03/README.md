# 📈 FinanceLink - 데이터 기반 금융상품 비교 서비스

사용자의 자산 데이터와 시장 지표를 결합하여 최적의 금융상품을 제안하고, Gemini AI 상담을 통해 의사결정을 돕는 반응형 웹 애플리케이션입니다.

---

## 🛠️ 기술 스택
* **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
* **Framework**: Bootstrap 5.3.3 (Responsive Grid System)
* **AI Integration**: Google Gemini 1.5 Flash API
* **Icons**: Bootstrap Icons

---

---
## 파일정보
* **chat.js**: 챗봇 구현 파일
* **main.html**: 메인화면 페이지
* **user_info.html**: 기본정보 수정 페이지
* **portfolio.html**: 포트폴리오 수정 페이지
---


## ✨ 주요 기능 및 UX 설계

### 1. 사용자 중심 네비게이션
* **접근성 강화**: 사용자가 가장 빈번하게 사용하는 **마이페이지** 버튼을 네비게이션 바와 모바일 토글 메뉴 옆에 배치하여 접근 단계를 최소화했습니다.
* **직관적인 홈 이동**: 프로필 및 포트폴리오 관리 페이지 상단 우측에 **Home** 버튼을 상시 배치하여 메인 화면으로의 복귀를 용이하게 설계했습니다.

### 2. 시장 데이터 인사이트 (NFLX 분석)
* **관심 종목 연동**: 사용자의 관심사인 **넷플릭스(NFLX)** 주가 지표를 메인에 시각화하고, 실시간 분석 페이지(Google Finance)로 연결되는 '더보기' 기능을 구현했습니다.
* **금융상품 큐레이션**: 이달의 인기 예적금 리스트를 제공하며, '상세보기' 클릭 시 각 은행의 공식 상품 안내 페이지로 연결됩니다.

### 3. 반응형 디자인 구현 (F305)
* **그리드 시스템**: 화면 가로 사이즈가 **768px 미만**으로 내려가면 투자 성향 섹션이 자기소개 섹션 밑으로 자동 배치되는 유연한 레이아웃을 구축했습니다.



### 4. Gemini AI 금융 가이드
* **실시간 상담**: Gemini 1.5 Flash 모델을 연동하여 사용자의 금융 관련 질의에 즉각 응답하는 플로팅 챗봇 위젯을 구현했습니다.

## 🔍 트러블슈팅 (Troubleshooting)

### 1. 레이아웃 간섭 및 그리드 깨짐
* **문제**: 외부 CSS의 `flex: 1` 설정이 부트스트랩의 `col-md-6` 클래스와 충돌하여 모바일 환경에서 세로 배치가 정상적으로 작동하지 않았습니다.
* **해결**: 레이아웃을 강제하는 기존 flex 속성을 제거하고 부트스트랩의 그리드 시스템에 전적으로 의존하도록 코드를 개편했습니다.

### 2. 페이지 전환 및 경로 인식 오류
* **문제**: 포트폴리오 수정 페이지(`test2.html`)에서 기본 정보 페이지(`user_info.html`)로 복귀할 때 404 오류가 발생했습니다.
* **해결**: 대소문자를 포함한 파일명 일치 여부를 전수 점검하고 `href` 경로를 상대 경로로 명확히 수정하여 해결했습니다.

### 3. UI 요소의 미세 위치 정렬
* **문제**: 상단 Home 버튼이 헤더 영역의 경계선에 걸쳐 시각적으로 부자연스러운 현상이 발견되었습니다.
* **해결**: CSS `position: absolute`와 `top: 8px` 설정을 통해 버튼을 파란색 헤더 안쪽 상단으로 안정적으로 이동시켰습니다.

## 📝 프로젝트 후기
* **디자인 통일성의 중요성**: 페이지별로 파편화된 HTML 구조 속에서 헤더 스타일과 홈 버튼 아이콘 위치 등을 일관되게 유지하는 과정이 까다로웠으며, 공통 CSS 관리의 필요성을 절감했습니다.
* **사용자 경험(UX) 고려**: 단순히 기능을 나열하기보다 마이페이지 버튼의 위치, 분석 데이터 더보기 버튼의 배치 등 사용자의 시선 흐름을 고려한 설계가 서비스의 완성도를 높인다는 것을 배웠습니다.
* **확장성**: 추후 실제 금융 API를 연동하여 더 정확한 실시간 데이터를 제공하고, 사용자 정보를 받아 맞춤형 금융리포트 제공하고 싶습니다.
